# Blackhuman Tap

## How do I install these formulae?

`brew install blackhuman/tap/<formula>`

Or `brew tap blackhuman/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "blackhuman/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
